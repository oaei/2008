Ontology Alignment Evaluation Initiative - OAEI-2008 Campaign

Conference track (Consensus workshop)

Conference track contains quite real-case ontologies suitable for ontology mappings task because of their heterogenous character of origin. There are no reference alignments, therefore there are employed methods of a posteriori evaluation There are two possible tasks related to this track, please see section Modalities.

Data sets

The collection of ontologies is dealing with conference organisation. They have been developed within OntoFarm project. 15 ontologies are included. Each of them is shortly described in the following table. Ontologies have been based upon three types of underlying resources:

  • actual conference (series) and its web pages (indicated in column Type with mark Web),
  • actual software tool for conference organisation support (indicated in column Type with mark Tool),
  • experience of people with personal participation in organisation of actual conference (indicated in column Type with mark Insider).
Name is derived from name of conference or conference organisation tool. For downloading this ontology, just click on its name.
DL expressivity was obtained from SWOOP tool which is Hypermedia-based OWL Ontology Browser and Editor.

Modalities

This year, there are two tasks related to this conference track. Participants can take part in either of them or in both of them. We ask participants to let us know to which task their results are related when they send us their results according to the schedule (see below).

  • generic correspondences - it means that participants can find out correspondences without any specific application context given,
  • application-specific correspondences - in this case, participants should find out correspondences with regard to application scenario - transformation application. It means that final correspondences are to be used for conference data transformation from one software tool for organizing conference to another one.

We will evaluate results of participants with several evaluation methods.
